1
Pick your Discord channels on the right and hit Save
2
Click ⬇ Download config.yml below — your Guild ID is already inside
3
Upload NinjuPlugin.jar to Aternos → Files → plugins/
4
Upload the downloaded config.yml to plugins/NinjuPlugin/ (create the folder if it doesn't exist)
5
Start the server — done! ✅ No manual editing needed

# Generated by NinjuBot Dashboard — no manual editing needed!

webhook_url: "https://ninjubot.onrender.com/minecraft/webhook"
guild_id: "YOUR_GUILD_ID"

events:
  chat:  true
  join:  true
  leave: true
  death: true
  start: true
  stop:  true

timeout_ms: 5000
