# NinjuBot Dashboard

A beautiful neumorphic dashboard for NinjuBot — a Discord bot by sdb_darkninja.

## 🌐 Live Site
Visit: https://sohamdasbiswas.github.io/NinjuBot-Dashboard

## 📦 Pages
- `index.html` — Landing page
- `dashboard.html` — Control panel / stats

## 🛠️ Setup
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Site will be live at `https://yourusername.github.io/NinjuBot-Dashboard`

## 🔗 Bot API
Update `BOT_API` in `app.js` to point to your bot's Render URL.
